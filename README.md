# 豆包无水印图片提取工具

一个浏览器扩展工具，用于提取豆包网站生成的无水印图片。

## 功能特点

- 🎯 **自动检测**: 自动检测豆包生成的图片，无需手动操作
- 💎 **无水印下载**: 提取并下载高质量的无水印图片
- 🖼️ **预览选择**: 提供友好的图片预览界面，支持批量选择
- 📝 **提示词显示**: 显示每张图片的生成提示词
- 🔄 **实时监听**: 实时监听API响应，自动更新图片列表

## 安装方法

### Chrome/Edge浏览器安装

1. 下载本项目并解压
2. 打开Chrome浏览器，进入扩展管理页面：`chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择解压后的文件夹
6. 安装完成！

### 其他Chromium内核浏览器

类似步骤，进入相应浏览器的扩展管理页面进行安装。

## 使用方法

1. 安装扩展后，访问豆包网站 [www.doubao.com](https://www.doubao.com)
2. 生成图片后，页面顶部会出现"下载"按钮
3. 点击"下载"按钮，弹出图片选择窗口
4. 在弹窗中浏览图片，选择需要下载的图片
5. 点击"下载"按钮保存无水印图片到本地

## 技术原理

本工具通过以下方式实现无水印图片提取：

1. **网络请求拦截**: 拦截豆包网站的图片生成API请求
2. **响应数据解析**: 解析API响应中的无水印图片URL
3. **用户界面创建**: 在页面添加下载按钮和图片选择界面
4. **文件下载**: 通过浏览器扩展API下载图片

### 关键代码说明

- `inject.js`: 注入到页面中的脚本，拦截API请求
- `content.js`: 内容脚本，负责UI创建和用户交互
- `background.js`: 后台脚本，处理文件下载
- `manifest.json`: 扩展配置文件

## 注意事项

- 本工具仅支持豆包网站生成的图片
- 需要登录豆包账号才能正常使用
- 生成的图片将保存在浏览器默认下载位置
- 请遵守豆包网站的使用条款和版权规定

## 故障排除

### 扩展不工作

1. 检查是否正确安装了扩展
2. 确认浏览器是否为最新版本
3. 刷新豆包页面后重试
4. 查看浏览器控制台是否有错误信息

### 无法下载图片

1. 检查浏览器是否允许扩展下载文件
2. 确认下载文件夹是否有写入权限
3. 尝试使用其他浏览器

### 找不到下载按钮

1. 确认已在豆包网站生成了图片
2. 刷新页面后重试
3. 检查控制台日志，确认是否成功拦截到API
